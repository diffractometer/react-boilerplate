# React Boilerplate

### Warning: NOT RELEASED

This is some React Boilerplate.

Shoutout to [react-slingshot](https://github.com/coryhouse/react-slingshot "Title") where most of the code and structure is stolen from.


## Installation

To install run

```sh
npm install
```

To run in dev mode

```sh
npm run dev
```

To build production

```sh
npm run build
```

To test production

```sh
npm start
```
